Page Replacement

FIFO:

Replace oldest page. Simple but suffers from Belady's Anomaly.

LRU (Least Recently Used):

Replace page not used for longest time. Better performance.

Optimal:

Replace page not needed for longest in future. Theoretical best.

Performance: Optimal < LRU < FIFO (page faults)

Programmed I/O:

CPU polls device status continuously. Simple but wastes CPU time.

Interrupt I/O:

Device interrupts CPU when ready. CPU can do other work meanwhile.

1. Issue command 2. CPU does other work 3. Device interrupts 4. CPU executes ISR 5. Resume work

DMA (Direct Memory Access):

Special controller transfers data without CPU.

1. CPU initializes DMA 2. DMA takes bus control 3. Transfers entire block 4. Sends interrupt when done

Modes: Burst (fastest), Cycle Stealing, Transparent
